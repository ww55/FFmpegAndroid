# FFmpegAndroid
android端基于FFmpeg库在中的使用。<br>
基于ffmpeg3.2.4版本，编译生成libffmpeg.so文件。<br>
目前音视频相关处理：<br>

- #### 音频剪切、拼接
- #### 音频混音
- #### 音频转码
- #### 音视频合成
- #### 音频抽取
- #### 音频解码播放
- #### 视频抽取
- #### 视频剪切
- #### 视频转码
- #### 视频截图
- #### 视频转GIF动图
- #### 视频添加水印
- #### 图片合成视频
- #### 视频解码播放
- #### 本地直播推流
- #### 实时直播推流
- #### 音视频解码播放
- #### OpenGL+GPUImage滤镜
